# How to Use this project:
1. Copy these requirements in reuirements.txt.
    asgiref==3.6.0
    certifi==2022.12.7
    charset-normalizer==3.1.0
    Django==4.1.7
    idna==3.4
    requests==2.28.2
    sqlparse==0.4.3
    tzdata==2023.3
    urllib3==1.26.15
2. Now in terminal type pip3 intstall -r requirements.txt
3. cd clearsky/skyclear
4. python manage.py runserver
